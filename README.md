Short audio sharing app. 


## Technologies

- **Rails:** 7.1.3.4
- **Ruby:** 3.2.2

### Installation

1. **Clone the repository:**
    ```sh
    git clone git@github.com:vigetlabs/short-audio.git
    cd shortaudio
    ```

2. **Install requirements:**

    ```sh
    pip install -r requirements.txt
    ```

3. **Apply migrations:**

    ```sh
    python manage.py migrate
    ```

3. **Run server:**

    ```sh
    python manage.py runserver
    ```
