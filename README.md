Short audio sharing app. 


## Technologies

- **Django:** 5.0.7
- **Python:** 3.12.4

### Installation

1. **Clone the repository:**
    ```sh
    git clone git@github.com:vigetlabs/short-audio.git
    cd shortaudio
    ```

2. **Install requirements:**

    ```sh
    pip install -r requirements.txt
    ```

3. **Apply migrations:**

    ```sh
    python manage.py migrate
    ```

3. **Run server:**

    ```sh
    python manage.py runserver
    ```
